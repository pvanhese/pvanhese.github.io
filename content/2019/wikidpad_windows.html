Run WikidPad From Source on Windows

Oct 01, 2019   How To   WikidPad   English

Run WikidPad From Source on Windows¶

WikidPad is a single user desktop wiki. If you want to run the most recent version of WikidPad or if you want to contribute to WikidPad, run it from source. This is how you do it.

Tested on Windows 7 64-bit.

Short version¶

Get the source from GitHub, add a copy of sqlite3.dll, pip install wxPython, and run with python WikidPad.py.

Install¶

These steps will install WikidPad to your home directory. Adapt if you want to install WikidPad in another directory.

① Make the installation folder:

> cd /D %userprofile%
C:\Users\user> mkdir WikidPad

② (optional, but recommended) Install git.

③ Get the source with git:

C:\Users\user> cd WikidPad
C:\Users\user\WikidPad> git clone https://github.com/WikidPad/WikidPad.git

Or (if you did not install git) download the source code (zip file) and unzip to C:\Users\user\WikidPad.

④ Install Python 3.

Note

Install Python 3.7.3 if you want to run the test suite using pytest (optional), because pytest fails with Python 3.7.4 (will be fixed with release of Python 3.7.5).

⑤ (optional, but recommended) Create a Python virtual environment and activate it:

C:\Users\user\WikidPad> python -m venv venv
C:\Users\user\WikidPad> venv\Scripts\activate

⑥ Install wxPython with pip:

C:\Users\user\WikidPad> pip install wxPython

⑦ Download sqlite3.dll and put a copy of it in a directory included in PATH (e.g., C:\Users\user\WikidPad\venv\Scripts if you created a virtual environment):

> copy sqlite3.* C:\Users\user\WikidPad\venv\Scripts

These steps will result in the following directory structure:

C:\Users\user\WikidPad            # installation directory
    |-- venv                      # Python virtual environment
        |-- Scripts
            |-- activate.bat
            |-- python.exe
            |-- sqlite3.dll
            |-- sqlite3.def
            |-- ...
        |-- ...
    |-- WikidPad                  # WikidPad source code
        |-- WikidPad.py
        |-- ...

Run¶

① (if you created a virtual environment) Activate the virtual environment:

> cd /D %userprofile%\WikidPad
C:\Users\user\WikidPad> venv\Scripts\activate

② Start WikidPad:

> cd /D %userprofile%\WikidPad\WikidPad
C:\Users\user\WikidPad\WikidPad> python WikidPad.py

Enjoy!
