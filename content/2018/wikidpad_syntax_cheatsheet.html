WikidPad Markup Cheat Sheet¶

This is a cheat sheet for WikidPad text formatting.

Headings¶

+ Header 1
++ Header 2
+++ Header 3
++++ Header 4

Phrase Emphasis¶

_italic_
*bold*

Horizontal Rules¶

----

Escaping markup¶

<< escaped text, this is a section of _text_ with no *formatting* >>

<<hide
this text is hidden
>>

# This is a comment line, not shown in preview.

Lists¶

Unordered list:
    * A list item
    * A second list item

Ordered list:
    1. Foo
    2. Bar

Autonumbered list:
    # First item
    # Second item
    # Third item

Preformatted Blocks¶

<<pre
Shown in monospace with
    exactly the same formatting
        as written
>>

Wiki Word Links¶

CamelCase

[Bracketedwikiword]

[noncamelcase]

[Wikiword|title in preview]

URL Links¶

Link to website: https://www.duckduckgo.com

Link to website with title: [https://www.wikipedia.org/|Wikipedia]

(Absolute) link to local file: [file:///E:/readme.txt]

(Relative) link to local file: [rel://../directoryabove/readme.txt]

Images¶

Supported image formats: JPG, GIF, PNG, TIF, BMP.

[file:///C:/Documents%20and%20Settings/example_image.bmp]

[rel://../path/to/image.jpg]

Tables¶

<<|
Column Heading 1  |  Column Heading 2
col 1             |  col 2
*bold*            |  _italic_
>>

Insertions¶

Table of Contents: [:toc:]

Backslash Escape Characters¶

Escape markup characters with a backslash ("\\") to display them as is.

\\  backslash

\*  asterisk

\_ underscore

\[ \] square brackets

Document Attributes¶

[attribute: value]

[icon: book]

[global.wrap: 80]

[alias: AliasName]
