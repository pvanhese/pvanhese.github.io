Code I Wrote¶

Fix for WikidPad¶

Fix for a long standing problem (since 2006) in WikidPad where renaming of a wiki word failed to correctly update all links to it. The problem was that WikidPad used a simple search and replace method to update the links, instead of using the AST and a text generator to properly update the wiki links.

My fix (2017) is included in WikidPad-2-3-beta16 and later.

It taught me (among other things): the internals of WikidPad pages, subpages and wiki links; the implementation of the wiki languages using pyparser; how to modify the wiki links in the AST, and how to build a text generator that takes the modified AST as input and returns the wiki text with correctly updated links as output, with the syntax according to the wiki language; the use of pytest to build a test suite.
